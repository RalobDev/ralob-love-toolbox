# Ralob Love Toolbox

A collection of reusable **Lua** libraries and modules designed for use with **LÖVE / Love2D**.

This repository centralizes utilities, systems, and abstractions developed across personal game projects. Its main goal is to provide a modular and reusable toolbox that helps speed up development while keeping the codebase clean, understandable, and easy to extend.

## 🎯 Purpose

This toolbox is mainly intended for personal use, but the modules are written with reusability in mind. Each library can be used independently whenever possible, making it easier to reuse systems across different Love2D projects.

## 📦 What you’ll find here

> Not every library has the same level of abstraction. Some are small utilities, while others are more complete systems.

- [Viewport](libs/Viewport/README.md) - A virtual resolution and scaling system that keeps rendering consistent across different screen sizes.
- [Class](libs/Class/README.md) - A lightweight class system for Lua, with support for inheritance, mixins, class comparison, and Lua LSP annotations.
